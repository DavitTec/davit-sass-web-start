# Module Commentary Tags

To add code comments into website web pages, you can use comments in HTML, JavaScript, and other programming languages to improve code readability, document functionality, and aid in debugging. In HTML, comments are written using `<!--` to start and `-->` to end the comment, and they are ignored by web browsers during rendering. For our added use we use an extra prefix character to its title.  To indicate a insertion of section ("§"), component (TBD) or partial  (TBD)
